Modules in NodeJS

tyeps of modules in nodejs
1.core module: module provide by node js to us
2.local module: module created by developer for their use in project 
3.thirdparty module: module created and developed by third party vendor, to use it first we have to install it then we can use it.

*core module: 
    os: it give information about operating system
    path: it return information aout file or directory path.
    _dirname: it gives current directory address
    _filename: it gves current filepath.
